Input and output formats for SNDlib


On this page, you find the necessary formats for specifying network planning instances and their solutions.

Each SNDlib instance consists of two input files:

  1. A network file describing the network (nodes, links, demands, and admissible paths), and
  2. a network model file describing the capacity and routing model, whether links are directed or undirected, etc.
Furthermore, there is a solution format for specifying feasible solutions. For all three file types, two possible formats are provided: a native ASCII format and an XML format. You can choose either of them.



  • Native ASCII:
    • Format conventions
    • Network model file: format description, example
    • Network file: format description, example
    • Solution file: format description, example (with detailed cost analysis)
  • XML:
    • HTML documentation
    • Network model file: XML schema file, example
    • Network file: XML schema file, example
    • Solution file: XML schema file, example
